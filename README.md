# ポモドーロタイマー

シンプルなブラウザ用ポモドーロタイマーです。フレームワーク不要、HTML / CSS / JavaScript のみで動作します。

## 機能

- 作業(25分)・小休憩(5分)・長休憩(15分)の3モードを切り替え可能
- 円形プログレスリングで残り時間を視覚的に表示
- 開始・一時停止・リセット操作
- 完了したポモドーロ(作業セッション)の回数を記録
- セッション終了時に通知音を再生

## 使い方

1. `index.html` をブラウザで開く
2. 上部のタブでモードを選択
3. 「開始」ボタンでタイマーを開始、「一時停止」で停止、「リセット」で初期値に戻す

## ファイル構成

- `index.html` — マークアップ
- `style.css` — スタイル
- `script.js` — タイマーのロジック
